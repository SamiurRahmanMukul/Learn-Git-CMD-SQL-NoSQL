* CMD Command:
	1. Color choose: 	  													$ color\?
	2. Select any color: 													$ color 0A
	3. Clear screen:	  													$ clear / cls

	4. Check current folder files: 								$ dir
	5. Hidden file open: 													$ dir/a
	6. Go folder: 																$ cd // (Folder name)
	7. Create a new folder: 											$ mkdir // (Created folder name)

	8. Back previous folder: 					   					$ cd ..
	9. Back previous or 2 or any previous folder: $ cd ../..

	10. Delete empty folder: 	  									$ rmdir // (Delete folder name)
	11. Delete not empty folder: 									$ rmdir /s // (Delete folder name)

	12. Shortly open any folder or files: 				$ // ( Only pressed Tab button any one select)
	13. Shortly open any folder or files: 				$ cd // ( Then press Tab button any one select)
